Portfolio Website

A personal portfolio website created to showcase my projects, skills, and growth as a developer.
The website includes a homepage, projects section, about page, and contact section.

Features
Responsive navigation bar
Project showcase section
Case study section
Background images and styled layouts
Footer with social links
Simple and clean UI
Multiple pages connected with navigation links
Built With
HTML5
CSS3
Vanilla JavaScript
Purpose of the Project

The purpose of this project was to practice building a complete multi-page website using only front-end technologies. It also helped me improve my understanding of layout design, navigation, styling, positioning, and responsive structure.

Challenges Faced

One challenge I faced was styling the layout correctly and making sure different sections aligned properly. Another challenge was working with background images and positioning text outside of image containers while still keeping the design clean.

I also struggled with organizing the pages and linking everything together at first, but through practice I became more comfortable with structuring projects.

What I Learned

Through this project I learned:

How to structure a multi-page website
How to style layouts using CSS
How to use Flexbox for positioning
How to create navigation between pages
How to add backgrounds, buttons, and project sections
How to improve the overall user experience with cleaner styling
